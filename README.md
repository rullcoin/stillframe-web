GitHub Pages should publish from `main` at `/`.

Live URLs:

- `https://rullcoin.github.io/stillframe-web/`

## Update rule

This repo is the live public legal/support surface for App Store, Play Store, and in-app Settings links. Keep it synchronized with the private app repo.

Update this repo whenever `silhouette-frame` changes public-facing name or brand, privacy claims, data collection or transmission, third-party processors, AI request payloads or disclosures, purchase/subscription behavior, crash or detection diagnostics, account/cloud/export/share behavior, HealthKit/Health Connect behavior, permissions, privacy manifests, store listing copy, support contact details, or public URLs.

Do not ship release-facing app changes while `docs/privacy-policy.md`, `docs/store-privacy-answers.md`, App Store / Play copy, Settings links, and these public pages disagree.
